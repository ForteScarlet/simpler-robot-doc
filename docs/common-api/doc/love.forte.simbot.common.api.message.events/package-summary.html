love.forte.simbot.common.api.message.events 0.0.1 API

See: Description

  • Interface Summary 
    Interface Description
    MsgGet监听消息的父接口。所有的监听消息都应当实现的容器：
    MessageEventGet与消息有关的事件MessageEventGet中除了需要实现MsgGet以外, 还要实现FlagContainer以标识一个消息内容的标识。 但是一般来讲, FlagContainer.flag 都可以用 id来代替。因此 flag提供为默认方法并使用id作为返回值。如果有特殊需要则重写
    MessageRecallEventGet与消息撤回有关的事件, 例如 群消息撤回 或者 私聊撤回一般来讲应该可以得到撤回的消息内容以及撤回时间
    IncreaseEventGet与 增加 有关的事件，例如 群友增加 或者 好友增加 增加事件除了存在 事件主体账号信息 以外, 还应存在 操作性账号信息.
    ReduceEventGet与 减少 有关的事件，例如 群友减少 或者 好友减少 增加事件除了存在 事件主体账号信息 以外, 还应存在 操作性账号信息.
    FriendIncrease好友增加.
    GroupMemberIncrease群友增加.
    PrivateMsg私聊消息.
    GroupMsg群消息， 需要实现 MessageEventGet , GroupContainer , PermissionContainer
    PrivateMsgRecall私聊消息撤回
    GroupMsgRecall群聊消息撤回
    FriendReduce好友减少事件
    GroupReduce群友减少事件
